Síntesis

El propósito de este vocabulario es el proveer las definiciones de las estructuras de los diferentes cubos de datos para las emisiones de gases de efecto invernadero. Para ello se reutiliza el vocabulario W3C Data Cube y se han generado instancias para cada estructura con la finalidad de indicar las dimensiones y medidas que se emplean en cada uno de los cubos. Adicionalmente, se reutilizan varios vocabularios y listas de códigos que permiten describir los detalles de los emisores de los gases y sus diferentes sectores
