Introducción volver a índice

Tras el desarrollo del vocabulario para la representación de datos de emisiones de gases de efecto invernadero (GEI), en este vocabulario se presentan las estructuras de los cubos para la representación de los datos agregados de las emisiones. Los cubos se definen mediante los conceptos y propiedades descritos en el vocabulario W3C Data Cube. Concretamente, se han generado las instancias que recogen los detalles necesarios para representar la estructura de cada cubo. Asimismo, para representar las dimensiones y medidas que se requieren se han reusado y definido varios vocabularios y listas de códigos. Este vocabulario ha sido desarrollado en el contexto del proyecto "Madrid Green".
