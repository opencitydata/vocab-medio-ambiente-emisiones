Introduction back to ToC

After the development of the vocabulary for the representation of greenhouse gas (GHG) emissions data, this vocabulary presents the structures of the cubes for the representation of the aggregated emissions data. Cubes are defined using the concepts and properties described in the W3C Data Cube vocabulary. Specifically, the instances have been generated that collect the details necessary to represent the structure of each cube. Also, to represent the dimensions and measures that are required, several vocabularies and code lists have been reused and defined. This vocabulary has been developed in the context of the "Madrid Green" project.
